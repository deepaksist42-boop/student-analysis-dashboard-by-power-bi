# 📊 Student Performance Dashboard

## 📌 Project Overview

This project presents an interactive dashboard designed to analyze student academic performance. It helps identify top-performing students, evaluate subject difficulty, and understand the impact of attendance on marks.

The dashboard enables educators and administrators to make data-driven decisions to improve student outcomes.

---

## 🎯 Objectives

* Identify top-performing students
* Analyze subject-wise performance
* Evaluate the relationship between attendance and marks
* Understand distribution of student scores
* Provide interactive filtering for deeper insights

---

## 📂 Dataset Information

The dataset used in this project includes the following fields:

* Student Name
* Class
* Subject
* Marks
* Attendance (%)
* Gender
* Exam Date
* Teacher

---

## 📊 Dashboard Features

### 🔹 KPI Cards

* Average Marks
* Highest Marks
* Lowest Marks
* Average Attendance

### 🔹 Visualizations

* 📊 Bar Chart – Average marks by subject
* 🏆 Student Ranking Table / Chart
* 🔵 Scatter Plot – Attendance vs Marks
* 📉 Histogram – Marks distribution
* 📈 Line Chart – Performance over time
* 🥧 Pie Chart – Gender distribution

### 🔹 Filters (Slicers)

* Class
* Subject
* Teacher
* Gender

---


## 🛠 Tools & Technologies

* Data Visualization Techniques

## 💡 Key Insights

* Students with higher attendance tend to score better
* Certain subjects have consistently lower average marks, indicating difficulty
* Top-performing students can be easily identified using ranking visuals
* Performance trends can be tracked over time
